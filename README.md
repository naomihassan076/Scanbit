# 📱 Scanbit - Smart QR Rewards System

> 🎯 **Physical scans unlock token incentives** - A decentralized QR code scanning system that rewards users with tokens for scanning physical QR codes in the real world.

## 🌟 Overview

Scanbit bridges the physical and digital worlds by creating a token-based incentive system for QR code scanning. Businesses, event organizers, and individuals can create QR codes with token rewards, encouraging real-world engagement and interaction.

## ✨ Key Features

- 🏆 **Token Rewards**: Earn SCANBIT tokens by scanning QR codes
- 🎯 **Smart Limits**: Daily scan limits and cooldown periods prevent abuse
- 📊 **User Statistics**: Track your scanning history and reputation
- ⏰ **Flexible Expiry**: QR codes can have custom expiration times
- 🏪 **Business Integration**: Perfect for marketing campaigns and customer engagement
- 🔒 **Anti-Spam Protection**: Built-in mechanisms to prevent gaming the system
